# ShopVerse Ecommerce Platform

Modern multi-page ecommerce platform developed with HTML, CSS, and JavaScript.

## Overview

ShopVerse is a frontend ecommerce web application designed to simulate the experience of a modern online marketplace. The project focuses on responsive design, interactive user interfaces, product browsing, and multi-page navigation using vanilla JavaScript.

The application provides users with various ecommerce features such as product exploration, category-based shopping, favorites management, shopping cart simulation, seller center access, notifications, customer support pages, and responsive layouts.

## How to Use

1. Open the homepage.
2. Browse products through categories or the explore page.
3. Search products using the search bar.
4. Add products to the shopping cart.
5. View saved products in favorites.
6. Access seller center and customer service pages.
7. Navigate through the website using the available menu options.
